### Changed

- **Build Optimizations**: Implemented multi-stage Docker builds for the backend.
- **Ollama Deployment**: Optimized Docker caching by moving Ollama installation to the final stage of the build process.

## [2.1.0] - 2026-03-08

### Added

- **PostgreSQL Migration**: Fully migrated DB from `better-sqlite3` to PostgreSQL using Prisma ORM.
- **Enterprise Security**: Added MFA (Multi-Factor Authentication), Email Verification, and Password Reset flows.
- **Dockerization**: Full stack Dockerization with persistent storage and integrated Ollama.
- **CI/CD Pipeline**: Enhanced GitHub Actions with Docker build validation and automated Prisma generation.
- **Service Layer**: Robust service architecture for AI (Gemini), SMS (Twilio), Voice (Bland), and CRM (Twenty).

### Changed

- **Codebase Reorganization**: Cleaned up the root directory and moved the backend entry point to `server/index.ts`.
- **API Refactoring**: All endpoints rewritten to support asynchronous Prisma queries.
- **Frontend Enhancements**: Updated user settings and account management to support new security features.

### Fixed

- Prisma Client generation issues and TypeScript synchronization between frontend and backend.
- Root directory clutter (removed legacy log and temporary files).
